Digital Thermometer with Overheat Alert System

PIC16F877A-based temperature monitoring system with ADC processing and UART data logging.

Objective

The objective of this project was to design a real-time digital thermometer capable of accurately monitoring temperature and triggering an alert when the temperature exceeds a predefined threshold of 80°C, ensuring system safety and reliability.

Background & Motivation

Temperature monitoring is critical in embedded and industrial systems where overheating can lead to failure or damage. This project focuses on reliable analog signal processing, precise ADC conversion, and robust alert mechanisms using a resource-constrained microcontroller.

Technical Architecture

  • PIC16F877A microcontroller
  • LM35 analog temperature sensor
  • 16×2 LCD display
  • CP2102 USB–UART interface

System Workflow

  • Analog Signal Acquisition — LM35 generates voltage proportional to temperature
  • ADC Processing — PIC ADC converts analog voltage into digital temperature values
  • Real-Time Display — temperature continuously displayed on LCD
  • Overheat Detection — alert triggered when temperature exceeds 80°C
  • Data Logging — temperature values transmitted to PC via UART

Firmware Development

  • ADC driver for analog-to-digital conversion
  • UART driver for serial communication
  • LCD driver for real-time display
  • Threshold detection and alert logic

Firmware was developed using MPLAB X and XC8, following an interrupt-driven design approach for reliable operation.

Testing & Validation

  • Temperature accuracy verified across multiple ranges
  • Calibration performed using reference voltages
  • Sensor linearity tested and validated
  • UART communication tested at multiple baud rates

Results & Evaluation

The system produced stable and accurate temperature readings throughout testing. The overheat alert mechanism functioned reliably, and UART-based data transmission remained robust during extended operation.

Tools & Technologies: PIC16F877A, Embedded C, MPLAB X, XC8, LM35, UART Communication
