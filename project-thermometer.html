Digital Thermometer with Overheat Alert System

PIC16F877A-based temperature monitor with ADC processing and UART output.

Objective

To develop a precise real-time temperature monitoring system with an alert mechanism when temperature exceeds 80°C, ensuring stability and reliability.

Technical Architecture

  • PIC16F877A Microcontroller
  • LM35 Temperature Sensor
  • CP2102 USB-UART
  • 16×2 LCD Display

System Workflow

  1. LM35 produces analog voltage
  2. PIC ADC reads and converts to °C
  3. LCD shows temperature
  4. Alert is triggered if temperature > 80°C
  5. UART logs data to PC

Firmware Development

  • ADC driver
  • UART driver
  • LCD driver
  • Threshold detection

Developed with MPLAB X + XC8 using interrupt-driven design.

Testing & Validation

  • Temperature accuracy tested across multiple ranges
  • Calibration using reference voltages
  • Sensor tested for linearity
  • UART tested at different baud rates

Results & Evaluation

The thermometer produced stable and precise readings, and the overheating alert worked reliably. UART communication was robust throughout extended testing.
