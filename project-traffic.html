Traffic Accident Prediction Using Machine Learning

A data-driven ML system for predicting accident severity and casualties.

Objective

The goal was to build a machine learning system capable of predicting accident severity and expected casualties using large-scale accident data. The system helps transport departments identify high-risk factors and apply targeted safety interventions.

Background & Motivation

Accident severity depends on multiple interacting factors—weather, lighting, surface conditions, and driver demographics. Traditional statistical methods struggle with non-linear relationships, so this project explored whether ensemble ML models could provide better predictive reliability and actionable insights.

Dataset

12,317 accident records with 15 input variables:

  • Driver profile (age, education, experience)
  • Road environment (surface condition, lanes, traffic density)
  • Weather and lighting condition
  • Collision type and number of vehicles

Targets:

  • Accident Severity — 0 (low), 1 (medium), 2 (high)
  • Number of Casualties — continuous value

Preprocessing

  • Missing/outlier handling
  • One-hot encoding for categorical fields
  • Train/test split (80/20)
  • Normalization for regression targets
  • Undersampling for balancing where needed

Modelling Approach

1. Severity Classification

  • Model: Random Forest Classifier
  • Hyperparameters: n_estimators, max_depth, max_features
  • Metrics: Accuracy, Precision, Recall, F1-score, Confusion Matrix

2. Casualty Regression

  • Model: Random Forest Regressor
  • Metrics: RMSE, MAE, R² score

Tools used: Python, Pandas, NumPy, Sciki
