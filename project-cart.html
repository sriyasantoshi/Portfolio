Smart Shopping Cart System

RFID-based automated billing system with wireless updates and real-time feedback.

Objective

The objective of this project was to design and implement a smart shopping cart capable of automatically detecting items, maintaining a running bill, and transmitting cart data wirelessly to reduce checkout delays and improve the retail shopping experience.

Background & Motivation

Conventional retail billing systems rely heavily on manual barcode scanning, which causes long queues and inefficiencies. By integrating RFID technology with embedded systems and wireless communication, this project explores a scalable approach to faster, user-friendly billing automation.

System Design

  • STM32F401 (ARM Cortex-M4) microcontroller
  • RC522 RFID reader (SPI interface)
  • HC-05 Bluetooth module (UART)
  • 16×2 LCD display (I²C)
  • RGB LED indicators and buzzer for feedback

Core Functional Modules

  • Automated Item Detection — RFID-based product identification when items are added or removed
  • Real-Time Cart Management — dynamic update of item count, pricing, and total cost
  • Wireless Communication — cart data transmitted to a desktop application via Bluetooth
  • User Feedback System — LCD display, LEDs, and buzzer for instant user interaction

Desktop Application

A desktop interface built using Python and PyQt5 to visualize the cart state, display detected items, compute the total price, and simulate the checkout process in real time.

Firmware Architecture

  • RFID handling and tag validation module
  • Cart state and pricing management logic
  • Bluetooth communication protocol
  • LCD, LED, and buzzer driver modules

Testing & Performance

  • Low-latency RFID scanning and detection
  • Stable SPI and UART communication
  • Immediate LCD updates on item changes
  • Reliable wireless data transfer

Evaluation

The system successfully demonstrated end-to-end automation of item detection and billing. It highlights strong embedded system design, real-time communication, and hardware–software integration suitable for IoT-based retail applications.

Tools & Technologies: STM32CubeIDE, Embedded C, RFID (RC522), Bluetooth (HC-05), Python, PyQt5
