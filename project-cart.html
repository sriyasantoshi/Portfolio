Smart Shopping Cart System

RFID-based automated billing with wireless updates and LCD feedback.

Objective

To design a smart retail cart that detects products automatically, maintains an internal inventory, and sends real-time updates wirelessly to reduce billing delays and enhance customer experience.

Design Overview

  • STM32F401 ARM Cortex-M4
  • RC522 RFID Reader (SPI)
  • HC-05 Bluetooth (UART)
  • LCD 16×2 (I2C)
  • RGB LED + Buzzer

Core Functional Modules

  1. Automated Item Detection via RFID scanning
  2. Real-Time Cart Management: updates LCD, pricing, quantity
  3. Wireless Data Transfer to PC via HC-05
  4. User Interface with indicators + buzzer

Desktop Application

Built using Python + PyQt5 to visualize cart state, show total price, and simulate checkout.

Firmware Architecture

  • RFID handler
  • Cart management module
  • Bluetooth communication module
  • LCD and feedback drivers

Testing & Performance

  • Fast RFID scanning
  • Stable SPI + UART communication
  • Immediate LCD updates
  • Low-latency wireless transfers

Evaluation

The system successfully automated item detection and billing while demonstrating strong embedded programming, IoT system design, and hardware–software integration.
