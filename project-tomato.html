Tomato Leaf Disease Detection Using CNN

Deep learning–based classification system using VGG16/VGG19.

Overview

Tomato crops suffer from a range of leaf diseases that reduce yield and create economic losses. This project builds a CNN-based image classifier using transfer learning (VGG16/VGG19) to automatically detect and classify 11 tomato leaf diseases.

Objective

To create an automated system that classifies tomato leaf images into 11 categories using deep learning, enabling accurate and early disease detection for smart agriculture.

Dataset & Problem Scope

  • Source: Kaggle – Tomato Disease Multiple Sources
  • Total Images: 32,535
  • Train/ Test Split: 80% / 20%
  • 11 classes including Late Blight, Early Blight, Leaf Mold, Mosaic Virus, Healthy, etc.

System Pipeline

  1. Image resizing & normalization
  2. Data augmentation (rotation, flip, zoom, shear)
  3. VGG16/19 convolution base for feature extraction
  4. Custom dense layers + dropout
  5. Training for 10 epochs
  6. Evaluation on unseen test images

Technical Approach

Model Architecture: VGG-based CNN with custom classification head.

  • Loss: Categorical Cross-Entropy
  • Optimizer: Adam / SGD
  • Metric: Accuracy
  • Softmax output with 11 neurons

Evaluation & Performance

Tested on unseen images with strong accuracy and stable validation performance. Transfer learning improved generalization, and augmentation reduced overfitting.

Prediction Workflow

  1. Input leaf image
  2. Resize + normalize
  3. Model inference
  4. Output predicted disease class
