Tomato Leaf Disease Detection Using CNN

Deep learning–based image classification system using transfer learning with VGG16 and VGG19.

Objective

The objective of this project was to develop an automated image classification system capable of identifying and classifying tomato leaf diseases into 11 categories using convolutional neural networks, enabling early and accurate disease detection in agricultural settings.

Background & Motivation

Tomato crops are highly susceptible to a variety of leaf diseases that significantly reduce yield and crop quality. Manual inspection is time-consuming and error-prone. This project leverages deep learning and transfer learning techniques to provide a scalable and reliable solution for smart agriculture.

Dataset

  • Source: Kaggle – Tomato Disease (Multiple Sources)
  • Total Images: 32,535
  • Classes: 11 (including Early Blight, Late Blight, Leaf Mold, Mosaic Virus, Healthy, etc.)
  • Train/Test Split: 80% / 20%

System Pipeline

  • Image resizing and normalization
  • Data augmentation (rotation, flipping, zoom, shear)
  • Feature extraction using VGG16/VGG19 convolutional base
  • Custom dense layers with dropout regularization
  • Model training and validation
  • Evaluation on unseen test images

Technical Approach

  • Architecture: VGG-based CNN with custom classification head
  • Loss Function: Categorical Cross-Entropy
  • Optimizer: Adam / SGD
  • Evaluation Metric: Accuracy
  • Output Layer: Softmax with 11 neurons

Evaluation & Performance

The model achieved strong classification accuracy on unseen test images with stable validation performance. Transfer learning significantly improved convergence speed, while data augmentation helped reduce overfitting.

Prediction Workflow

  • Input tomato leaf image
  • Image preprocessing and normalization
  • Model inference using trained CNN
  • Predicted disease class output

Tools & Technologies: Python, TensorFlow/Keras, VGG16, VGG19, OpenCV, NumPy
