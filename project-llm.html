Hybrid Text Summarization System with Comparative Model Analysis

Comparative analysis of a fine-tuned LLaMA model and a custom-built LLM using social media and news data.

Objective

The objective of this project was to develop an automated NLP system capable of aggregating large-scale textual data from social media platforms and news sources, generating concise summaries and sentiment insights while comparing transformer-based fine-tuned models against custom language model architectures.

Background & Motivation

With the rapid growth of unstructured text data across platforms like Twitter, Reddit, and online news portals, manual analysis becomes impractical. This project explores whether parameter-efficient fine-tuning techniques such as LoRA can outperform traditional or lightweight custom models in real-world summarization and sentiment analysis tasks.

Technology Stack

  • LLaMA with LoRA (PEFT)
  • PyTorch & Hugging Face Transformers
  • Twitter API (Tweepy)
  • Reddit API (PRAW)
  • NewsAPI / GNews
  • FastAPI backend
  • VADER sentiment analysis
  • NLTK & BeautifulSoup

Core Functional Modules

  • Multi-Source Data Retrieval — automated collection of data from social media and news APIs
  • Text Preprocessing Pipeline — cleaning, normalization, and separation of social and news content
  • AI-Powered Summarization — fine-tuned LLaMA model enhanced with LoRA adapters
  • Sentiment & Trend Analysis — polarity scoring and topic trend identification using VADER

Model Architecture

A dual-model approach was implemented. A fine-tuned LLaMA model using LoRA adapters was trained on curated news datasets and adapted to social media text. Its performance was benchmarked against a custom-built language model to evaluate efficiency, accuracy, and scalability.

System Pipeline

  • API aggregation layer (Twitter, Reddit, News)
  • Text preprocessing and normalization module
  • LLaMA inference engine with LoRA adapters
  • Custom baseline model for comparison
  • Sentiment analysis and trend detection
  • FastAPI-based REST interface

Testing & Performance

  • Robust API rate limiting and error handling
  • Efficient parameter-efficient fine-tuning workflow
  • Accurate real-time sentiment scoring
  • Comparative evaluation using summarization quality metrics

Evaluation

The system successfully demonstrated scalable multi-source data aggregation, high-quality abstractive summarization, and effective sentiment analysis. It highlights strong expertise in modern NLP, transfer learning, and production-ready API design.

Tools & Technologies: PyTorch, Transformers, LLaMA, LoRA (PEFT), FastAPI, Twitter API, Reddit API, NewsAPI, VADER
