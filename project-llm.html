ONGOING PROJECT

Hybrid Text Summarization System

An evolving NLP system focused on multi-source text aggregation and intelligent summarization using Large Language Models and parameter-efficient fine-tuning.

NLP LLMs LLaMA LoRA PyTorch FastAPI

Project Status

üöß This project is currently under active development.
The system is being designed, prototyped, and iteratively improved with a focus on scalability, model efficiency, and real-world applicability.

Initial Project Objective

The goal of this project is to build a hybrid text summarization platform that can:

  • Collect textual data from multiple sources such as news articles and social media
  • Generate concise, coherent summaries using Large Language Models
  • Compare LLM-based summaries with a lightweight baseline model
  • Provide sentiment insights alongside generated summaries

Planned System Architecture

Data Pipeline (Planned)

  • Content ingestion from news APIs and social platforms
  • Text preprocessing: cleaning, normalization, and tokenization
  • Deduplication and basic entity extraction

Modeling Approach (In Progress)

  • LLaMA-based foundation model for abstractive summarization
  • LoRA adapters for parameter-efficient fine-tuning
  • Baseline transformer model for performance comparison
  • Separate sentiment analysis module using transformer classifiers

Backend & API (Planned)

  • FastAPI-based backend for model inference
  • REST endpoints for summarization and sentiment analysis
  • Dockerized deployment for reproducibility

Current Progress

  • Literature review on abstractive summarization and LoRA fine-tuning
  • Dataset exploration and preprocessing pipeline design
  • Initial experiments with transformer-based summarization models
  • Backend API structure and project scaffolding

Planned Evaluation

Once model training is completed, the system will be evaluated using:

  • ROUGE metrics for summarization quality
  • BERTScore for semantic similarity
  • Inference time and resource utilization
  • Qualitative comparison between baseline and LLM-based outputs

Future Roadmap

  • Complete LoRA fine-tuning on summarization datasets
  • Integrate sentiment analysis into the summarization pipeline
  • Add multi-source summarization support
  • Deploy a working demo with API access
  • Extend support to multilingual content
